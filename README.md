# Arihanta Naturals

Arihanta Naturals is a responsive, modern web application built for showcasing organic products, connecting with customers through a contact form, and presenting farm information with visuals and maps. The frontend is built with React, styled using CSS variables, and integrates a backend API for handling contact form submissions.
